# Security policy

Cutguard is a research-preview instruction repository. Report sensitive issues through a GitHub private security advisory when the repository is public, or email `lustresense@gmail.com`.

Do not open a public issue containing a real credential, an exploitable bypass, or private user data. Non-sensitive documentation and false-positive reports may use normal issues.

Useful reports include contradictory instructions that remove a required guard, adapter generation that changes security meaning, realistic credentials in examples, or a claimed secure example with a reproducible authorization bypass.
